# Workshop GitHub Collaboration - CONFLICT SOLVE

Workshop ini dirancang untuk mengajarkan best practices dalam berkolaborasi menggunakan GitHub. Peserta akan mempelajari:

- Dasar-dasar Git dan GitHub
- Branching dan merging strategies
- Pull requests dan code reviews
- Conflict resolution
- Team workflows dan CI/CD integration

Melalui sesi interaktif dan hands-on practice, peserta akan mampu bekerja secara efektif dalam tim development menggunakan GitHub sebagai platform kolaborasi utama.
